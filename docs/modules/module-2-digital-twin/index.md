# Module 2: Digital Twin (Gazebo & Unity)

## Overview
This module focuses on physics simulation and environment building. You'll learn to simulate physics, gravity, and collisions in Gazebo, create high-fidelity rendering for human-robot interaction in Unity, and simulate sensors like LiDAR, Depth Cameras, and IMUs.

## Learning Objectives
- Set up and work with Gazebo simulation environments
- Simulate physics, gravity, and collisions accurately
- Create high-fidelity rendering in Unity for human-robot interaction
- Simulate various sensors (LiDAR, Depth Cameras, IMUs) in both environments

## Prerequisites
- Completion of Module 1 (ROS 2 fundamentals)
- Basic understanding of 3D environments
- Familiarity with simulation concepts

## Technology Stack
- Gazebo Garden
- Unity 2022.3 LTS
- Sensor simulation tools
- Physics engines
